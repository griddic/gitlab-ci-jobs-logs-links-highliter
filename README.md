# gitlab-ci-jobs-logs-links-highliter
This script make all links in gitlab-ci jobs logs clickable.

It is a browser extension. This extension works for jobs in giltab-ci. It finds text wich looks like a link and make such text clickable like a link.

One may use http://www.tampermonkey.net as a user scripts manager. 
Direct link to install this user script is https://greasyfork.org/ru/scripts/396174-gitlab-jobs-links-highliter .
